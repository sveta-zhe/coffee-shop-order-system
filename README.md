# Система управления заказами для сети кофеен «Кофеин»

## О проекте
Это учебный проект для портфолио системного аналитика.  
Цель — спроектировать систему автоматизации приема заказов в кофейне, включая:
- регистрацию заказов
- интеграцию с платежным шлюзом
- отчеты для управляющего
- очередь заказов для бариста

## Бизнес-контекст
Сеть кофеен «Кофеин» сталкивается с проблемами:
- Длинные очереди в часы пик
- Ошибки при передаче заказов бариста
- Отсутствие оперативной отчетности для управляющих

**Главная цель MVP:** сократить время обслуживания клиента с 5 до 2 минут и снизить количество ошибок при передаче заказа до 0.
