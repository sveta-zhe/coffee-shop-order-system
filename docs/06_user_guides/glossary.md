# Глоссарий

| Термин | Определение |
| :--- | :--- |
| **API** | Интерфейс для общения программ между собой |
| **Бариста** | Сотрудник кофейни, который готовит напитки |
| **Дашборд** | Экран с отчетами и графиками |
| **Заказ** | Выбранные клиентом напитки и добавки |
| **Интеграция** | Связь нашей системы с другой (например, с платежным шлюзом) |
| **Очередь** | Список заказов в порядке поступления |
| **Приемочные тесты** | Проверка, что система работает так, как нужно заказчику |
| **Спринт** | Период работы команды (обычно 1–2 недели) |
| **Статус заказа** | Текущее состояние: new, paid, in_progress, ready, completed |
| **Управляющий** | Руководитель кофейни |
