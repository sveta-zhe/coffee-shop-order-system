# Руководство управляющего

## Как смотреть отчеты

### Дашборд управляющего
Откройте в браузере страницу дашборда: 

**Что вы видите на дашборде:**
- **Выручка за сегодня** — общая сумма всех оплаченных заказов
- **Количество заказов** — сколько заказов сделано
- **Средний чек** — выручка / количество заказов
- **Топ-3 напитка** — самые популярные позиции
- **Загрузка по часам** — график количества заказов

### Фильтры
Вы можете посмотреть данные за:
- Сегодня
- Вчера
- Текущую неделю
- Текущий месяц

### Детальный отчет по заказам
Нажмите **«Детальный отчет»** — откроется таблица со списком всех заказов:
- Номер заказа
- Дата и время
- Сумма
- Статус
- Способ оплаты

Вы можете выгрузить отчет в Excel, нажав **«Экспорт»**.

## Управление продуктами

### Как добавить новый напиток
1. Перейдите в раздел **«Управление меню»**
2. Нажмите **«Добавить продукт»**
3. Заполните поля:
   - Название (например, «Раф»)
   - Категория (кофе / добавка / десерт)
   - Цена
   - Количество в наличии
4. Нажмите **«Сохранить»**

### Как изменить цену
1. Найдите продукт в списке
2. Нажмите **«Редактировать»**
3. Измените цену
4. Нажмите **«Сохранить»**

### Как убрать напиток из меню (закончился в наличии)
1. Найдите продукт в списке
2. Нажмите **«Редактировать»**
3. Поставьте галочку **«Нет в наличии»**
4. Нажмите **«Сохранить»**

## Управление пользователями

### Как добавить нового бариста
1. Перейдите в раздел **«Пользователи»**
2. Нажмите **«Пригласить»**
3. Введите email сотрудника
4. Выберите роль: **бариста**
5. Нажмите **«Отправить приглашение»**

Сотрудник получит письмо со ссылкой для входа.

### Как удалить пользователя
1. Найдите пользователя в списке
2. Нажмите **«Удалить»**
3. Подтвердите удаление

> **Важно:** Удалить можно только пользователя, у которого нет активных заказов.

## Часто задаваемые вопросы

**Вопрос:** Почему выручка за сегодня не совпадает с наличными в кассе?  
**Ответ:** В дашборде отображаются все оплаченные заказы, включая онлайн-оплату. Для сверки с кассой используйте отчет **«Детальный отчет»** и фильтр по способу оплаты.

**Вопрос:** Как часто обновляются данные на дашборде?  
**Ответ:** Данные обновляются в реальном времени. При создании нового заказа дашборд меняется в течение 5 секунд.

**Вопрос:** Могу ли я посмотреть выручку за прошлый месяц?  
**Ответ:** Да, выберите в фильтре «Месяц» и укажите нужную дату.
