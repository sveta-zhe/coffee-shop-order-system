# Руководство бариста

## Как работает очередь заказов

### Экран очереди
На планшете открыт экран «Очередь заказов».  
Заказы отображаются в порядке поступления (сверху — самые старые).

**Карточка заказа содержит:**
- Номер заказа
- Напиток и добавки
- Время создания
- Кнопка **«Готово»**

### Что делать с новым заказом
1. Когда появляется новый заказ — он добавляется в конец очереди.
2. Начните готовить заказ в порядке очереди.
3. Когда заказ готов — нажмите **«Готово»** на его карточке.

### Как отменить заказ
Если клиент попросил отменить заказ:
1. Нажмите на карточку заказа
2. Выберите **«Отменить заказ»**
3. Подтвердите отмену

> **Важно:** Отменить можно только заказ со статусом «paid» или «in_progress».  
> Если заказ уже «ready» — отмена невозможна, клиент должен его забрать.

### Ошибки и их решение

| Проблема | Решение |
| :--- | :--- |
| Заказ не появляется в очереди | Проверьте соединение с интернетом. Обновите страницу. |
| Нажал «Готово» по ошибке | Отменить нельзя. Предупредите клиента, что заказ уже готов. |
| Клиент не забирает заказ | Через 15 минут система отправит повторное уведомление. Через 30 минут — заказ считается «просроченным». |
