# Сценарии поведения системы (Gherkin)

## 1. Клиент создает заказ

**Feature:** Создание заказа
**As a:** Клиент
**I want to:** выбрать напиток и добавки
**So that:** заказать кофе по своему вкусу

**Scenario:** Успешное создание заказа
- **Given** я нахожусь на экране выбора напитка
- **When** я выбираю "Латте"
- **And** добавляю "сироп"
- **And** нажимаю "В корзину"
- **Then** в корзине отображается "Латте с сиропом"
- **And** отображается общая сумма

**Scenario:** Ошибка при создании (ингредиент закончился)
- **Given** я нахожусь на экране выбора напитка
- **When** я выбираю "Раф"
- **But** ингредиента нет в наличии
- **Then** система отображает сообщение "Временно нет в наличии"
- **And** кнопка "Добавить" неактивна

---

## 2. Бариста видит очередь заказов

**Feature:** Отображение очереди
**As a:** Бариста
**I want to:** видеть список ожидающих заказов
**So that:** готовить в порядке поступления

**Scenario:** Новый заказ поступает в очередь
- **Given** на планшете бариста открыт экран очереди
- **When** клиент создает и оплачивает заказ
- **Then** на планшете появляется новая карточка заказа в конце списка
- **And** карточка содержит название напитка, добавки, время создания
