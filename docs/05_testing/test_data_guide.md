# Инструкция по загрузке тестовых данных

## Назначение
Тестовые данные нужны для:
- Ручного тестирования (проверить, что заказы отображаются)
- Автоматического тестирования (база не пустая)
- Демонстрации заказчику (показать, как выглядят данные)

## Требования
- Установлен PostgreSQL
- Создана база данных `coffee_shop`
- Выполнены скрипты создания таблиц (schema.sql)

## Как загрузить тестовые данные

### Способ 1. Через командную строку
```bash
psql -U postgres -d coffee_shop -f sql/test_data.sql
